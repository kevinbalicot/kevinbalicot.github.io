Features

Everything you need to build a marketplace

MMP will help your developers build the best marketplace. Based on Sylius, this plugin transforms a Sylius ecommerce into a powerful marketplace following Sylius's standards.

Tier Sellers

Create an unlimited number of sellers. Add a seller to products, build a dedicated page for a seller or use your default catalog...

Groups

Sellers can view their customers and are able to create customer groups including a default one.

Promotion, payment, shipment

Build your own rules or use discriminators for promotion, payment and shipment. Everything is possible!

Dedicated admin panel

Want to give access to your sellers? Use the dedicated plugin with a default admin panel, use our default panel or build your own!

API

All our entities are exposed through the standard Syius API built on top of Api Platform.

More!

We also use our plugin with our customers projects. Help us to build more features suiting your needs!

Frequently asked questions

Is it opensource?
At the moment, MMP is open for our partners and their customers. It will be available for everyone once our first roadmap is complete!
Price and license?
MMP is available through an EUPL license for community and a paid one for enterprises with dedicated support and strong commitments.
Sylius and PHP compatibility?
MMP is currently available with Sylius 1.11, and it uses PHP8. Our plugin follows the standard semantic versioning but the community version don't have any backward compatibility.
Release and roadmap?
We are working (hard) on the latest version for our first OSS release but our plugin is already used by many projects.
